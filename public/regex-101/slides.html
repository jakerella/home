Simple Character Test

/t/.test("Connect Tech");  // true
/z/.test("Connect Tech");  // false

Testing versus Matching

true/false vs. matched groups


/t/.test("Connect Tech");  // true
"Connect Tech".match(/t/);  // ["t"]

Where's the other "n"?


"Connect Tech".match(/n/);  // ["n"]

We need to make it global!


"Connect Tech".match(/n/g);  // ["n", "n"]

The "g" here is called a "flag", the "global" flag.

Other Flags

Other Flags


"Connect Tech".match(/t/);  // ["t"]
    

// case-insensitive AND global
"Connect Tech".match(/t/ig);  // ["t", "T"]

There's also a "multi-line" flag: m
